# Lockedin: Behavioral Protocol & High-Stakes Accountability

**Lockedin** is a high-fidelity behavioral bank designed to transform intention into obligation. By staking real capital on personal goals and utilizing social verification, users create a "commitment protocol" where failure has immediate financial and reputational consequences.

## 🚀 Core Philosophy
Most productivity tools track progress; **Lockedin enforces it.** We believe that habits are weak until they are enforced by risk. Our system utilizes "Pain Tiers" and social oversight to ensure that the cost of failure is higher than the effort of execution.

## 🛠 Tech Stack
- **Frontend:** React, TanStack Start (File-based Routing), Tailwind CSS v4.
- **Backend:** Convex (Real-time Database, Server Functions, Auth).
- **Icons:** Lucide React.
- **Animations:** Framer Motion.
- **Identity:** Google OAuth + BVN Verification (Identity Anchoring).

## 💎 Features
- **Vault Protocols:** Stake NGN (Kobo) on specific mandates (Fitness, Habits, etc.).
- **Pain Tiers:** 
  - *Serious:* 2% daily penalty for missed execution.
  - *Locked In:* 5% daily penalty for missed execution.
- **Witness Network:** Connect with accountability partners who verify your evidence.
- **The Reward Pool:** High-integrity users earn dividends from the lack of discipline in others.
- **Integrity Score:** A dynamic reputation metric (0-100%) that tracks your protocol adherence.

## 🏗 Infrastructure
- **Midnight Sweep:** Automatic daily check for missed check-ins.
- **Payment Bridge:** Integrated infrastructure for Paystack/Flutterwave funding.
- **Evidence Engine:** Secure photographic evidence storage via Convex File Storage.
- **Real-time Engine:** Instant synchronization of streaks, balances, and notifications.

## 📈 Business Model
Lockedin operates on a high-margin behavioral model:
- **60% Protocol Fee:** Collected from forfeited stakes to fund operations and growth.
- **30% Community Dividend:** Redistributed to high-integrity users to incentivize long-term adherence.
- **10% Liquidity Reserve:** System security and insurance fund.
