# Lockedin: Product Development Roadmap

This roadmap outlines the evolution of Lockedin from a behavioral protocol to a global performance bank.

## Phase 1: Identity & Protocol Core (Completed)
- [x] High-fidelity Auth (Google OAuth).
- [x] Mandatory BVN Verification (Nigerian Identity Anchored).
- [x] Protocol Vault Creation (Staking Logic).
- [x] Community Witness Discovery Interface.
- [x] Basic Integrity Score Engine.

## Phase 2: The Execution Engine (Current)
- [x] **Evidence Engine:** Integration of Convex File Storage for photographic proof.
- [x] **The Midnight Sweep:** Daily cron jobs for automated penalty detection.
- [x] **Dividend Distribution:** Automated weekly reward engine for high-integrity users.
- [x] **Payment Bridge Infrastructure:** Ready for Paystack/Flutterwave API keys.
- [ ] **Mobile Optimization:** Full PWA (Progressive Web App) compliance for home-screen usage.

## Phase 3: Financial Scalability (Q2 2024)
- [ ] **Multi-Currency Support:** Support for USD/GBP via Stripe/LemonSqueezy.
- [ ] **Tiered Subscriptions:** Premium tiers with lower protocol fees but higher dividend multipliers.
- [ ] **Corporate Goals:** B2B version for high-performance teams (Sales/Engineering).
- [ ] **Automated Disputes:** AI-powered verification of evidence to reduce partner workload.

## Phase 4: Social & AI Performance (Q3 2024)
- [ ] **AI Performance Coach:** LLM-powered insights based on check-in consistency.
- [ ] **Guilds/Sectors:** Users can join "Clans" to compete in collective staking challenges.
- [ ] **Public Leaderboards:** Global rankings for Integrity Scores.
- [ ] **The Lockedin Card:** Debit card that only unlocks funds if you hit your weekly streak.

- [ ] Implement end-to-end encryption for private evidence logs.
